Genymotion google map howto

Select a device which uses android version 4.2.2 and above, start the device using genymotion button.

enter 
(AndroidSDKpath)\android-sdk\platform-tools

>adb install com.android.vending-20130716.apk
>adb install com.google.android.gms-20130716.apk

Drag Genymotion-ARM-Translation_v1.1 to the Virtual machine (yes, onto the face).
Click yes all the way.

Restart the virtual machine.

And run your android app, you should be able to see the google map
